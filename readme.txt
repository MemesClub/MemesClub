Разработать статический сайт "Зоопарк" с помощью языка HTML, содержащий 10-15 страниц. В качестве редактора использовать HTML-редактор, например NotePad++. На сайте представить информацию по следующим раздела:

- история зоопарка;

- основные сведения о зоопарке (адрес, площадь, количество животных и т.д.);

-информация о животных, упорядоченная по видам;

- режим работы зоопарка;

- предоставляемые услуги (кормление животных, катание, фотографирование и т.д.).

Остальные разделы на усмотрение разработчика.

Необходимо использовать все основные теги для форматирования абзацев, разделов, заголовков, а также использовать на страницах различные элементы дизайна - цвет, изображения, видео.

Отчет по лабораторной работе должен содержать:

- описание логической структуры сайта (взаимосвязь разделов);

- исходный текст страниц, снабженный комментариями;

- скрин-шоты страниц.

Варианты заданий.

1. Московский зоопарк

2. Зоопарк Санкт-Петербурга
